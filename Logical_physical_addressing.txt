:: یک کامپیوتر برای اتصال به شبکه نیاز به کارت شبکه دارد
:: کارت شبکه میتواند بیسیم یا سیم دار باشد
:: Mac Address | Physical Address -> هر کارت شبکه یک آدرس فیزیکی دارد که شرکت سازنده بر روی آن هک کرده است
:: هر کامپیوتر برای اتصال به شبکه علاوه بر آدرس فیزیکی نیازمند یک آدرس منطقی هم است
:: آدرس فیزیکی در لایه دوم مدل ا اس آی قرار میگیرد (Data Link)
:: آدرس فیزیکی در لایه سوم مدل ا اس آی قرار میگیرد (network)
:: آدرس فیزیکی به صورت پیش فرض ثابت است
:: آدرس منطقی بسته به شبکه میتواند عوض شود
:: IP Address -> منظور آدرس منطقی است
:: Mac Address -> منظور آدرس فیزیکی است
:: IPv4 , Ipv6 -> آدرس آیپی دارای دو نسخه است
:: [Mac Address] -> (48 bit base 16 Hexadecimal) | [IPv4 Address] -> (32 bit base 10 decimal) | [] [IPv6 Address] -> (128 bit base 16 Hexadecimal)
:: هر کامپیوتر برای اتصال به شبکه به هر دو آدرس آیپی نیاز دارد
:: تعداد آدرس های فیزیکی هر کامپیوتر برابر است با تعداد کارت شبکه های متصل شده به آن کامپیوتر
:: تعداد آدرس آیپی های منطقی میتواند به ازای هر کارت شکه نا محدود باشد
:: Mac -> media access control
:: OUI -> organization unique identifier -> شناسه منحصر به فرد سازمانی
:: برای اینکه تشخیص بدیم چند تا کارت شبکه روی سیستم است مرحل زیر را طی میکنیم => [
    1) Win + R -> این دکمه ها را میزنیم 
    2) ncpa.cpl -> را وارد میکنیم
    3) لیست کارت شبکه های موجود را به ما نشان میدهد و با دابل کلیک بر روی هر کدام در قسمت دیتیل میتوان اطلاعات و آیپی آن کارت شبکه را مشاهده کرد
]

:: اضافه کردن آدرس آیپی ورژن ۴ با کمک ترمینال -> [
    1) getmac -> در ترمینال با بالاترین سطح دسترسی عبارت را وارد میکنیم
    // لیست کارت های شبکه را نشان میدهد 
    2) ipconfig -> لیست آیپی های موجود بر هر کارت شبکه را نشان میدهد 

]

:: برای دیدن کارت شبکه های کامپیوتر خود باید مراحل زیر را طی کنیم => [
    1) دکمه ویندوز + آر را میزنیم 
    2) سپس دستور ncpa.cpl را وارد میکنیم 
    3) وارد محیطی میشویم که کارت های شبکه را به ما نشان میدهد 
    // دیدن کارت های شبکه از طریق دستور getmac در ترمینال 
    // میتوان با وارد کردن آدرس مک در وبسایت زیر نوع کارت شبکه را مشخص کرد 
    (https://macvendors.com/)
    (https://www.macvendorlookup.com/)
]

:: تغییر آدرس مک سیستم عامل => [
    1) devmgmt.msc -> دیوایس منیجر سیستم را به ما نشان میدهد 
    2) network adaptor -> به این تب میرویم که لیست کارت شبکه های ما را نشان میدهد
    3) کارت شبکه مورد نظر را انتخاب کرده و روی آن دابل کلیک میکنیم
    4) advance -> وارد این تب میشویم 
    5) localy adminstratory address -> را انتخاب میکنیم و از سمت راست در قسمت value مقدار آدرس جدید را برای آن قرار میدهدیم
]


----------------------------- ساختار مک درس -------------------------------
:: این نوع آدرس ۴۸ بیتی است و ساختاری مشابه به زیر را دارد 
:: 00-00-00-00-00-00 -> ff-ff-ff-ff-ff-ff -> میتواند در بازه مقابل قرار بگیرد
:: سه قسمت اول آدرس مک نشان دهنده OUI است 
// به این معنی که مشخص میکند که این کارت شبکه توسط کدام شرکت تولید شده است
:: سه قسمت دوم منحصر به فرد دستگاه میباشد

----------------------------- ساختار ipv4 -------------------------------
:: این آدرس از ۴ قسمت تشگیل شده است و ۳۲ بیت است
:: 0.0.0.0 -> 255.255.255.255 -> میتواند در بازه عددی زیر قرار بگیرد
